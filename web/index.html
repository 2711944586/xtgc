Method notes

研究对象是一个系统，不是一串孤立算法。

系统边界从“单航班是否延误”扩展到“机场网络如何恢复”。网页保留现场检查点，报告展开公式、假设和复现路径。

风险预测

模型输出计划阶段到达延误概率，实际起降时间和延误原因不进入训练，避免信息泄漏。

pi = P(yi=1 | xi) yi = I(ArrDel15i=1)

网络传播

机场小时状态由传播矩阵、恢复控制、外部冲击和误差共同驱动，传播矩阵只保留网络邻接关系。

x(t+1) = A x(t) + B u(t) + G w(t) + epsilon(t) rho(A) < 1

多准则评价

成本型指标正向化后组合 AHP 主观权重与熵权客观权重，再计算 TOPSIS 接近度。

w = lambda wAHP + (1-lambda) wEntropy Ci = Di- / (Di+ + Di-)

结论边界

推荐策略随情景、偏好权重和损失矩阵改变，最终结论写成条件判断而不是绝对最优。

s* = argmaxs in S score(s | scenario, w) L(s) = sumk P(k) loss(s,k)

Closing brief

恢复优先时选动态组合，成本保守时保留基准策略。
